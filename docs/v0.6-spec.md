## Role

Implement Composite Memory MCP Server v0.6.

and maintainable MCP memory platform.

# Phase 4 --- Ingestion Pipeline

Architecture:

    MCP remember
          |
          v
    Validation
          |
          v
    Canonical SQL Store
          |
          v
    Outbox
          |
          +--> Vector indexing
          |
          +--> Graph indexing

Do not write directly to multiple stores without transaction tracking.
